# La structure du projet

- assets : dossier utilisé pour mettre les documents (images, ...) 
- components : dossier pour mettre les components.
- styles : pour mettre les .css
## How to

Installer les dépendances avec:

### `npm install`

Exécuter le projet :
### `npm start`

Puis click sur :
### `http://localhost:3000`
